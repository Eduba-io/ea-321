# Thor Dynamics × Eduba landing page

Single-page pitch prepared by Eduba for the Thor Dynamics team at eMerge Americas 2026 (Booth 367).

## Serve locally
```
cd "website html"
python3 -m http.server 8080
```
Open http://localhost:8080/

## Files
- `index.html` - page markup
- `styles.css` - brand-token CSS
- `logo.png` - Thor Dynamics webclip mark (only logo asset publicly reachable)
